# Super Simple Highlighter

## What is it?

A Chrome extension for highlighting text, and automatically attempt to restore highlight on each page revisit.

## Installation

打开Chrome浏览器的插件页面 拖入SuperSimpleHighlighter.crx

## 颜色分级

- Purple : 顶级  
- Blue   : 重要 Class Function
- Yellow : 提示 String
- Cyan   : 有用 Variable
- Green  : 正确 Right
- Orange : 警告 Warning
- Red    : 错误 Error
- Gray   : 否定
